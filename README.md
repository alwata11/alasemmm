# شركة العاصم للاستثمارات العقارية والسياحية
## Al-Asim Real Estate & Tourism Investment Company

### Website Files
- `index.html` — Home page
- `about.html` — About Us
- `vision.html` — Vision & Mission
- `values.html` — Our Values
- `contact.html` — Contact Us
- `assets/css/style.css` — Global stylesheet
- `assets/js/main.js` — JavaScript (language toggle, dark mode, menu)
- `assets/images/logo.svg` — Logo placeholder (replace with `logo.png`)

### To Use Your Own Logo
Replace `assets/images/logo.svg` with your actual `logo.png` file and update
the `src` attributes in all HTML files from `logo.svg` to `logo.png`.

### Features
- Full Arabic (RTL) / English (LTR) bilingual toggle
- Dark / Light mode toggle
- Preferences saved in localStorage
- Responsive for desktop, tablet, and mobile
- Scroll-triggered reveal animations
- Sticky header with scroll effects
